Welcome to URL Shorten API

You can start anytime by typing /api/v1/ then the URL you want to be shorten.

Rules

  • Do not include http:// with your URL
  • Do not include www with your URL
We will take care of all of the above for you

CRUD

To perform CRUD with this api follow these rules:

CREATE link
/api/v1/url/:yourURL     | with GET method
GET ALL links
/api/v1/urls/     | with GET method
GET one link
/api/v1/urls/:urlID     | with GET method
UPDATE one link
/api/v1/urls/:urlID     | with POST method
DELETE one link
/api/v1/urls/:urlID     | with DELETE method
